# ⚽ football-lineup-generator
A winform application that generates a football lineup from a single line of text (e.g., 1-4-4-2). Not only does it generate the positions automatically, it also fill these positions with random players written in a .csv file. The players are picked based on their given position or very similar position. The application also includes .csv file creation and modification. The application use sofascore's position names.
# 💻 Technologies
- Visual studio 2026 community
- WinForms
- C#
# 📚 Features
- Ability to create, delete and modify .csv files directly with in the application.
- Lineup generation with a lineup visual (position name and player's name).
- Fill players in the lineup based on their position or based on similar positions.
